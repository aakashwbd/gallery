# Image Gallery

This project only for testing purpose.

## Run Locally

Clone the project

```bash
  git clone https://github.com/aakashwbd/gallery.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
